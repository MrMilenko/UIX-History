INSTALL
======================================================================
unrar and load files onto the root of C
X will bring up config panel to adjust settings
R+L+White = screenshot
R+L+Right = Screensaver
R+L+Left = HD INFO
black = memoryx
R+L+Up = Turn on BGM/Change ST
R+L+Down = Turn off BGM
X = config


BUGS/possible problems
======================================================================
In order for skinning to work properly there MUST BE A PRESET FOR THE SKIN!!!!  Pressing A on a skin in the menu without a patch will result in a ruined xbe.

You must use the included xbe for the patcher to work properly

The Preset, Screenshot (in screenshots.xip), and the Skin's xip must all be named the same thing

For all skins use the default.xap in the Skins provided...you must add
menubacking, chrome, bottom_1 - bottom_8, hilight, and cube xbxs

I know of a few existing bugs with the menus and patcher...an update will be released in the next few days
